Introduction to Coding

What is Coding?

Coding is a way of telling the computer what to do using special instructions.

Why is Coding Important?

  • Helps in problem-solving.
  • Creates games and apps.
  • Helps computers understand commands.

Basic Coding Concepts

  • Commands – Instructions given to a computer.
  • Sequences – Steps to follow in order.
  • Loops – Repeating a task multiple times.

Fun Coding Tools for Kids

  • Scratch – Create animations with blocks.
  • Code.org – Fun coding games.
  • Blockly – Drag-and-drop coding for kids.

Computer Ethics and Safety

Good Computer Habits

  • ✔ Sit properly while using the computer.
  • ✔ Take breaks after every 30 minutes.
  • ✔ Do not eat or drink near the computer.

Cyber Safety Rules

  • ✔ Do not open unknown emails or links.
  • ✔ Never share passwords.
  • ✔ Be kind online and avoid cyberbullying.

Respecting Others Online

  • Always use polite language.
  • Do not copy other people’s work without permission.
